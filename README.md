# 💻 GitHub Repository Explorer

🚀 A web application that allows users to search and explore GitHub repositories with sorting and filtering features.

---

## 📌 Project Description

This web application enables users to search for GitHub repositories based on keywords and explore useful details such as stars, language, and repository descriptions. It uses the GitHub REST API to fetch real-time data.

---

## 🎯 Purpose

The purpose of this project is to demonstrate:

* JavaScript fundamentals
* API integration using fetch
* Use of array higher-order functions (map, filter, sort)
* Responsive UI development

---

## 🌐 API Used

* GitHub REST API (Repository Search)
  https://api.github.com/search/repositories?q=javascript&sort=stars&order=desc&per_page=20&page=1

---

## ✨ Planned Features

* 🔍 Search repositories by keyword
* 🔃 Sort repositories by stars
* 🎯 Filter repositories by language
* 📄 Display repository details (name, description, stars)
* 📑 Pagination support
* ❤️ Mark favorite repositories
* 🌙 Dark mode toggle

---

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript (ES6+)
* Fetch API
