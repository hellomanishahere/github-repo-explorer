# 💻 GitHub Repository Explorer

🚀 A web application that allows users to search and explore GitHub repositories with sorting, filtering, and pagination features.

---

## 📌 Project Description

This web application enables users to search for GitHub repositories based on keywords and explore useful details such as stars, forks, language, and descriptions. It uses the GitHub REST API to fetch real-time data and provides an interactive UI for better user experience.

---

## 🔗 Live Demo
https://hellomanishahere.github.io/github-repo-explorer/

---

## 🎯 Purpose

The purpose of this project is to demonstrate:

- JavaScript fundamentals  
- API integration using fetch  
- Use of array higher-order functions (map, filter, sort)  
- Dynamic UI updates  
- Responsive web design  

---

## 🌐 API Used

- GitHub REST API (Repository Search)  
[https://api.github.com/search/repositories?q=example](https://api.github.com/search/repositories?q={search_term})

---

## ✨ Features

- 🔍 Search repositories by keyword  
- 🔃 Sort repositories by stars, forks, or last updated  
- 🎯 Filter repositories by language  
- 📄 Display repository details (name, description, stars, forks)  
- 📑 Pagination support (Next / Previous)  
- 🌙 Dark mode toggle  

---
